## рҹ‘Җ What is this?

Ever walked all the way to the NIT Srinagar hostel mess only to find out they're serving something you don't like? Or tried to complain about food quality and had no idea where to do it?

**Mess Management System** solves exactly that вҖ” and it's built **exclusively for NIT Srinagar students**. Only `@nitsri.ac.in` email addresses are allowed to register. It's a full-stack MERN web app that puts mess operations online вҖ” students can check menus, give feedback, raise complaints, and log in with OTP or Google. Admins get a dashboard to manage everything from one place.

> рҹҸ« **Institution:** National Institute of Technology, Srinagar (NIT Srinagar), J&K
> рҹ”’ **Restricted to:** `@nitsri.ac.in` email domain only
> рҹҢҗ **Built with:** React + Vite В· Node.js + Express В· MongoDB Atlas В· Firebase

## вҡЎ Quick Start

> Want to run this locally? Here's the short version:

```bash
# 1. Clone it
git clone https://github.com/prashantgoyal7691/mess-management.git
cd mess-management

# 2. Install server deps
cd server && npm install

# 3. Install client deps
cd ../client && npm install

# 4. Add your .env files (see Environment Setup below)

# 5. Start both servers
# Terminal 1 - backend
cd server && npm run dev

# Terminal 2 - frontend
cd client && npm run dev
```

App runs at `http://localhost:5173` вҖ” API runs at `http://localhost:5000`
